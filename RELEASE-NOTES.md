### v0.17.2
*(unreleased)*
* uses jcel 0.23.2
* uses the OWL API 4.1.3, and can be used as a plug-in for Protégé 5.0.0-beta-21
* build commands:
```
$ mvn clean install
```
* plug-in: `gel/target/de.tu-dresden.inf.lat.gel-0.17.2.jar`
* library: `gel/target/gel-0.17.2.jar`
